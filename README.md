# ReFind

ReFind is a Firefox add-on that builds upon the default find feature.

ReFind allows the user to perform a more advanced text search on a webpage. Beyond vanilla text searches, it can search based on regex patterns, and for combinations of words that appear in close proximity to one another.


## Installation

1. Download from [Firefox Add-ons](https://addons.mozilla.org/addon/refind)
2. Or load temporarily via `about:debugging`

## Usage

The toolbar can be opened with Ctrl+F. Subsequent/previous results can be navigatated via Enter/Shift+Enter respectively. Escape exits the toolbar.

## Features
- **Regex search**: Use regular expressions to find patterns
- **Proximity search**: Find words within N characters of each other
- **Match case**: Case-sensitive searching
